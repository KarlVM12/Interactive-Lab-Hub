# AI Interaction Logs

This directory contains automated logs of student interactions with WendyTA (GitHub Copilot Chat) organized by lab.

## Log Files

```
WendyTA/logs/
├── Lab1_ai_interaction_log.md
├── Lab2_ai_interaction_log.md
├── Lab3_ai_interaction_log.md
├── Lab4_ai_interaction_log.md
├── Lab5_ai_interaction_log.md
├── Lab6_ai_interaction_log.md
└── FinalProject_ai_interaction_log.md
```

## How It Works

- **One file per lab**: Each lab has a dedicated log file with boilerplate instructions
- **Automatic appending**: WendyTA adds timestamped entries when providing substantial help
- **Student identity**: Tracked via git commits (no names stored in logs)
- **Timestamp format**: `YYYY-MM-DD HH:MM:SS` for consistency

## For TAs

Review these logs to:
- Understand common student challenges per lab
- Track AI assistance patterns
- Identify areas where instruction could be improved
- Monitor appropriate AI usage in the course

## For Students

These logs are automatically created when you interact with WendyTA. Remember to:
1. Commit the log files along with your lab work
2. Document AI assistance in your README.md files
3. Follow proper attribution guidelines as outlined in the course syllabus
